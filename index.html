What This Dashboard Does

This dashboard uses LASSO regression to estimate FX fair values based on equity sector performance. It identifies which currencies are undervalued or overvalued relative to what equity market signals suggest.

Features:

  • Fair value estimates for 25+ currency pairs
  • Residual analysis (% deviation from fair value)
  • Sector coefficient breakdown for each currency
  • Interactive charts with historical trends
  • Daily automated updates
